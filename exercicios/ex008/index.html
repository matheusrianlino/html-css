Principais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag <b>, uma tag não semântica

Nesta frase temos um termo em destaque usando a tag <strong>, uma tag semãntica

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag <i>, uma tag não semântica

Nesta frase temos um termo em ênfase usando a tag <em>, uma tag semântica

Texto Marcado

Podemos criar tambem um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Podemos criar um texto grande e um texto pequeno usando as tags <big>(Tag obsoleta) e <small> respectivamente.

Texto deletado

Podemos marca um texto como excluido usando a tag <del>, indicando que deve ser lido, porem não considerado.

Texto Inserido

Podemos marcar um texto como inserido usando a tag <ins>, dando ênfase e indicando que foi adcionado depois.

Podemos sublinhar um texto utilizando tambem a tag <u>, uma tag não semântica.

Texto Sobrescrito

Podemos sobescrever textos usando a tag <sup>

Exemplo: x2

Texto Subescrito

Podemos subescrever textos usando a tag <sub>

Exemplo: H2O
